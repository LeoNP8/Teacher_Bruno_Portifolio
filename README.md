# Teacher Bruno - Portfolio

A modern, responsive portfolio website for an English teacher named Bruno. This project features a clean design, smooth animations, and a mobile-friendly layout.

## Features

- **Responsive Design**: Adapts seamlessly to different screen sizes (desktop, tablet, mobile).
- **Smooth Animations**: Uses Intersection Observer for elegant fade-in and slide-in effects.
- **Interactive Elements**: Includes a mobile menu toggle and subtle parallax effects.
- **Modern UI**: Features a clean layout with Google Fonts and custom styling.

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge).

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/LeoNP123/Teacher_Bruno_Portifolio.git
    ```
2.  Open `index.html` in your web browser.

## Usage

- Navigate through the sections using the menu or by scrolling.
- Click the "Contact" button to jump to the contact section.
- Use the floating WhatsApp button to easily get in touch.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Styling and animations.
- **JavaScript (Vanilla)**: Interactivity and scroll effects.
